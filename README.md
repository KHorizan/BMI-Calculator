# BMI Calculator

A simple BMI (Body Mass Index) Calculator built using React for practice.

## Features
- Enter your height and weight
- Calculates BMI instantly
- Displays BMI category (Underweight, Normal, Overweight, Obese)
- Simple and clean UI

## Tech Stack
- React
- Vite
- CSS
